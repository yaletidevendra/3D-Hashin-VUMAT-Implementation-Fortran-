# 3D-Hashin-VUMAT-Implementation-Fortran-
Fortran implementation of 3D Hashin damage model as a VUMAT for Abaqus/Explicit. Simulates progressive failure in composite materials, including fiber/matrix tension, compression, and shear. Enables high-efficiency 3D composite simulations with user-defined material properties.


Overview

This repository contains a Fortran implementation of a 3D Hashin damage model for composite materials, implemented as a VUMAT subroutine for Abaqus.

The model simulates progressive failure in composites under 3D loading, capturing multiple damage modes including:

Fiber tension and compression

Matrix tension and compression

Shear failure

It is suitable for explicit finite element simulations of laminated composites with user-defined material properties.
